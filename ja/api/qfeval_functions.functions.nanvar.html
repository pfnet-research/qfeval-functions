qfeval_functions.functions.nanvar

nanvar(x, dim=(), unbiased=True, keepdim=False)[ソース]

NaN値を無視してテンソルの分散を計算します。

この関数は、指定された次元に沿ってテンソル要素の分散を計算します。計算時には NaN 値は除外されます。分散は、不偏推定量（要素数 N-1 で除算）または偏推定量（要素数 N で除算）のいずれかを用いて計算できます。ここで N は NaN 値を除いた要素数です。

パラメータ:

  • x (Tensor) -- 入力テンソル。

  • dim (int or tuple of ints, optional) -- 分散を計算する次元を指定します。指定しない場合（デフォルトは空のタプル）は、すべての要素に対して分散を計算します。単一の次元または複数の次元を指定できます。

  • unbiased (bool, optional) -- デフォルト値の``True``の場合、ベッセル補正を適用し、非NaN要素の数を``N``として``N-1``で除算します。``False``の場合、``N``で除算します。

  • keepdim (bool, optional) -- True の場合、出力テンソルは入力テンソルと同じ数の次元を持ち、削減された次元のサイズは 1 になります。False の場合（デフォルト）、削減された次元は削除されます。

戻り値:

非NaN要素の分散。形状は dim と keepdim パラメータに依存します。

戻り値の型:

Tensor

サンプル

>>> x = torch.tensor([1.0, 2.0, nan, 4.0, 5.0])
>>> QF.nanvar(x)
tensor(3.3333)

>>> # With biased estimator
>>> QF.nanvar(x, unbiased=False)
tensor(2.5000)

>>> # 2D tensor with dimension specification
>>> x = torch.tensor([[1.0, 2.0, nan],
...                   [4.0, nan, 6.0]])
>>> QF.nanvar(x, dim=1)
tensor([0.5000, 2.0000])

>>> # Keep dimensions
>>> QF.nanvar(x, dim=1, keepdim=True)
tensor([[0.5000],
        [2.0000]])
