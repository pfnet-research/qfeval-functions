qfeval_functions.functions.bfill

bfill(x, dim=-1)[ソース]

指定された次元に沿って欠損値を後方補完します。

この関数は、指定された次元に沿って最後の有効な値（NaNでない値）を後方に伝播させます。各NaN値については、その次元に沿って後に現れる最も近い有効な値で置き換えられます。もしNaN値の後に有効な値がない場合、そのNaN値は変更されません。

パラメータ:

  • x (Tensor) -- 値を補完するための入力テンソル。

  • dim (int, optional) -- 後方補完を実行する次元。デフォルトは -1 で、最後の次元を指定します。

戻り値:

入力テンソルと同じ形状を持ち、可能な場合はNaN値をその直後の有効な値で置き換えたテンソル。

戻り値の型:

Tensor

サンプル

>>> x = torch.tensor([1.0, nan, nan, 4.0, 5.0])
>>> QF.bfill(x)
tensor([1., 4., 4., 4., 5.])

>>> x = torch.tensor([[nan, 2.0, nan, 4.0],
...                   [5.0, nan, nan, nan]])
>>> QF.bfill(x, dim=1)
tensor([[2., 2., 4., 4.],
        [5., nan, nan, nan]])
