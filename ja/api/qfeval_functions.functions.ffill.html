qfeval_functions.functions.ffill

ffill(x, dim=-1)[ソース]

指定された次元に沿って欠損値を前方補完します。

この関数は、指定された次元に沿って最後の有効な値（NaNでない値）を前方に伝播させます。各NaN値については、その次元に沿って直前に現れる最も新しい有効な値で置き換えられます。もしNaN値の直前に有効な値が存在しない場合、そのNaN値は変更されずにそのまま保持されます。

パラメータ:

  • x (Tensor) -- 値を補完するための入力テンソル。NaN値が含まれている場合があり、これらは指定された方法で置換されます。

  • dim (int, optional) -- 順方向フィルを実行する次元。デフォルトは -1 で、最後の次元を指定します。

戻り値:

入力テンソルと同じ形状で、NaN値は可能な限り直前の有効な値で置き換えられます。

戻り値の型:

Tensor

サンプル

>>> # Simple 1D forward fill
>>> x = torch.tensor([1.0, nan, nan, 4.0, 5.0])
>>> QF.ffill(x)
tensor([1., 1., 1., 4., 5.])

>>> # 2D example with dim=1
>>> x = torch.tensor([[nan, 2.0, nan, 4.0],
...                   [5.0, nan, nan, 8.0]])
>>> QF.ffill(x, dim=1)
tensor([[nan, 2., 2., 4.],
        [5., 5., 5., 8.]])

>>> # Forward fill along columns (dim=0)
>>> x = torch.tensor([[1.0, nan, 3.0],
...                   [nan, 2.0, nan],
...                   [4.0, nan, nan]])
>>> QF.ffill(x, dim=0)
tensor([[1., nan, 3.],
        [1., 2., 3.],
        [4., 2., 3.]])
