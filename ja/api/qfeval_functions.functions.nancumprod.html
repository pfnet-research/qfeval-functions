qfeval_functions.functions.nancumprod

nancumprod(x, dim)[ソース]

指定された次元に沿って累積積を計算し、NaN を 1 として扱います。

この関数は、指定された次元に沿って要素の累積積を計算します。ここで、NaN値は乗算の単位元である1として扱われます（累積積の計算目的のため）。ただし、元のテンソルでNaN値を持つ位置は、出力でもNaN値のままとなります。

パラメータ:

  • x (Tensor) -- 値を含む入力テンソル。

  • dim (int) -- 累積積を計算する次元を指定します。

戻り値:

入力テンソルと同じ形状のテンソルで、累積積算値を格納します。元のNaN位置はそのままNaNとして保持されます。

戻り値の型:

Tensor

サンプル

>>> # Simple cumulative product with NaN
>>> x = torch.tensor([2.0, nan, 3.0, 4.0])
>>> QF.nancumprod(x, dim=0)
tensor([ 2., nan,  6., 24.])

>>> # 2D tensor along columns
>>> x = torch.tensor([[1.0, 2.0, nan, 4.0],
...                   [2.0, nan, 3.0, 5.0]])
>>> QF.nancumprod(x, dim=1)
tensor([[ 1.,  2., nan,  8.],
        [ 2., nan,  6., 30.]])

>>> # Cumulative product along rows
>>> x = torch.tensor([[2.0, 3.0],
...                   [nan, 4.0],
...                   [5.0, 2.0]])
>>> QF.nancumprod(x, dim=0)
tensor([[ 2.,  3.],
        [nan, 12.],
        [10., 24.]])

>>> # Multiple NaNs
>>> x = torch.tensor([1.0, nan, nan, 2.0, 3.0])
>>> QF.nancumprod(x, dim=0)
tensor([1., nan, nan, 2., 6.])
