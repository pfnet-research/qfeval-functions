qfeval_functions.functions.nancovar

nancovar(x, y, dim=-1, keepdim=False, ddof=1)[ソース]

Compute covariance between two tensors, ignoring NaN values.

この関数は、指定された次元に沿ってテンソル x と y の共分散を計算します。計算対象から除外されるのは、いずれかの値が NaN であるペアです。共分散は、2つの変数がどの程度連動して変化するかを測る指標です。numpy.cov とは異なり、この関数は共分散行列を生成するのではなく、各バッチインデックスに対して要素ごとの共分散を計算します。

この関数はテンソルのブロードキャスト処理においてメモリ効率に優れていますが、数値精度よりもメモリ効率を優先するため、多くのNaN値が存在する場合には精度が約半精度まで低下する可能性があります。

NaNを考慮した共分散は以下のように計算されます：

\[\text{Cov}(X, Y) = \text{E}[(X - \mu_X)(Y - \mu_Y)]\]

ここで、期待値は有効な値のペア（NaNでない値）に対してのみ計算され、\(\mu_X\) および \(\mu_Y\) は有効な値に対して計算された平均値です。

パラメータ:

  • x (Tensor) -- 最初の入力テンソル。

  • y (Tensor) -- 第2の入力テンソル。 :attr:`x`とブロードキャスト可能である必要があります。

  • dim (int, optional) -- 共分散を計算する次元。デフォルトは -1 で、最後の次元を意味します。

  • keepdim (bool, optional) -- 出力テンソルが dim を保持するかどうか。デフォルトは False。

  • ddof (int, optional) -- 自由度のデルタ値。計算に使用される除数は N - ddof で、ここで N は有効な（NaNでない）ペアの数を表します。デフォルト値は 1 です。

戻り値:

有効な（NaNでない）ペアのみに対して計算された共分散値。形状は入力次元と keepdim パラメータに依存します。

戻り値の型:

Tensor

サンプル

>>> # Simple covariance with NaN values
>>> x = torch.tensor([1.0, 2.0, nan, 4.0, 5.0])
>>> y = torch.tensor([2.0, 4.0, 6.0, nan, 10.0])
>>> QF.nancovar(x, y, dim=0)
tensor(8.6250)

>>> # 2D tensors with NaN values
>>> x = torch.tensor([[1.0, nan, 3.0, 4.0],
...                   [5.0, 6.0, nan, 8.0]])
>>> y = torch.tensor([[2.0, 4.0, 6.0, nan],
...                   [10.0, nan, 14.0, 16.0]])
>>> QF.nancovar(x, y, dim=1)
tensor([4.0000, 9.1111])

>>> # Population covariance (ddof=0)
>>> x = torch.tensor([1.0, nan, 3.0, 4.0])
>>> y = torch.tensor([2.0, 4.0, nan, 8.0])
>>> QF.nancovar(x, y, dim=0, ddof=0)
tensor(4.5000)

>>> # With keepdim
>>> x = torch.tensor([[1.0, 2.0, nan],
...                   [4.0, nan, 6.0]])
>>> y = torch.tensor([[2.0, nan, 6.0],
...                   [8.0, 10.0, nan]])
>>> QF.nancovar(x, y, dim=1, keepdim=True)
tensor([[nan],
        [nan]])

警告

メモリ効率最適化のため、多くのNaN値を扱う場合、計算精度が若干低下する可能性があります（約半精度）。NaN値が多い場合により高い精度が必要な場合は、PyTorchのJITコンパイル機能を使用してCUDAカーネルを直接実行することを検討してください。
