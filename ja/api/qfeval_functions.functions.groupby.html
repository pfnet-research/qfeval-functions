qfeval_functions.functions.groupby

groupby(x, group_id, dim=-1, empty_value=nan)[ソース]

指定された次元に沿ってテンソル要素をグループ識別子でグループ化します。

この関数は、与えられたグループ識別子に基づいてテンソルの要素をグループに再編成します。各グループに属するすべての要素を含む新しい次元を作成します。これはSQLのGROUP BY句やpandasのgroupby操作に似ていますが、テンソル演算用に最適化されています。

The function adds a new dimension immediately after the specified dimension. This new dimension represents the items within each group. Since tensors require fixed-size dimensions, groups with fewer elements are padded with empty_value.

パラメータ:

  • x (Tensor) -- グループ化する入力テンソル。

  • group_id (Tensor) -- ``x.shape[dim]``と同じ長さの整数グループ識別子の1次元テンソル。各要素は、:attr:`x`の対応する要素が属するグループを指定します。グループIDは非負整数である必要があります。

  • dim (int, optional) -- 要素をグループ化する次元。デフォルトは-1（最後の次元）です。

  • empty_value (scalar, optional) -- グループサイズの最大値よりも要素数が少ないグループをパディングするために使用される値。デフォルトは nan です。

戻り値:

入力テンソルに1次元追加されたテンソルです。形状は:attr:x`と同じですが、次元:attr:`dim`は2つの次元に置き換えられます: ``(num_groups, max_group_size)`。要素はグループ所属に基づいて再配置されます。

戻り値の型:

Tensor

サンプル

>>> # Group a 1D tensor
>>> x = torch.tensor([10., 20., 30., 40., 50.])
>>> group_id = torch.tensor([0, 1, 0, 1, 0])
>>> grouped = QF.groupby(x, group_id)
>>> grouped
tensor([[10., 30., 50.],
        [20., 40., nan]])

>>> # Group along a specific dimension of 2D tensor
>>> x = torch.tensor([[1., 2., 3., 4.],
...                   [5., 6., 7., 8.]])
>>> group_id = torch.tensor([0, 1, 0, 2])
>>> grouped = QF.groupby(x, group_id, dim=1)
>>> grouped
tensor([[[1., 3.],
         [2., nan],
         [4., nan]],

        [[5., 7.],
         [6., nan],
         [8., nan]]])

>>> # Using custom empty value
>>> x = torch.tensor([1, 2, 3, 4, 5], dtype=torch.int)
>>> group_id = torch.tensor([0, 0, 1, 1, 1])
>>> grouped = QF.groupby(x, group_id, empty_value=-1)
>>> grouped
tensor([[ 1,  2, -1],
        [ 3,  4,  5]], dtype=torch.int32)
