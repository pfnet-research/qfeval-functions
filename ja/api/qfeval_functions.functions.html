関数

apply_for_axis

与えられたデータ x を、形状 (batch, n) のテンソルを期待する関数 f に適用します。

bfill

最後に有効な値を後方に伝播します。

bincount

指定された次元に沿ってテンソル内の各値の頻度を計算します。出力テンソルの最小長をサポートします。

bollinger_band

対応するボリンジャーバンドの上限/中央/下限値を返します。

correl

`x`と`y`の間のピアソン相関係数を返します。

covar

指定された次元に沿って、与えられたテンソル間の共分散を計算します。

cumcount

各グループ内の各項目に0から :math:`N-1`までの番号を割り当てます。ここで、:math:`N`はグループ内の項目数です。

eigh

固有値と固有ベクトルを計算します。

einsum

torch.einsumの型付きバージョン。

ema

指定されたテンソルの指数加重移動平均を返します。

ffill

有効値を前方に伝播します。

fillna

NaN、正の無限大、負の無限大の値を指定された数値で置き換えます。

gaussian_blur

指定された軸 dim に沿って、x に指定された sigma パラメータを持つガウシアンフィルタを適用します。

group_shift

指定されたマスクをスキップしながら、テンソルを指定された次元に沿ってシフトします。

groupby

group_id で x の項目をグループ化します。

ma

指定されたテンソルの移動平均を返します。

mmax

指定されたテンソルの移動最大値を返します。

mmin

指定されたテンソルの移動最小値を返します。

mstd

指定されたテンソルの移動標準偏差を返します。

msum

テンソルの移動和（スライディングウィンドウ和）を計算します。

mulmean

メモリ効率に優れた方法で (x * y).mean(...) を計算します。

mulsum

メモリ効率に優れた方法で (x * y).sum(...) を効率的に計算します。

mvar

指定されたテンソルの移動分散を返します。

nancorrel

`x`と`y`の間のピアソン相関係数を返し、NaNは無視します。

nancovar

与えられたテンソル間の共分散を計算します。

nancumprod

nancumsum

nankurtosis

各行の尖度を返し、非数値値（NaN）は無視します。

nanmax

指定された次元 dim において、入力テンソルの各行の最大値を含むタプル (values, indices) を返します。Not a Numbers (NaNs) は無視されます。

nanmean

各行の平均値を返し、Not a Numbers (NaNs) は無視します。

nanmin

指定された次元 dim において、入力テンソルの各行の最小値を含む名前付きタプル (values, indices) を返します。Not a Numbers (NaNs) は無視されます。

nanmulmean

メモリ効率に優れた方法で QF.nanmean(x * y, ...) を計算します。

nanmulsum

メモリ効率に優れた方法で QF.nansum(x * y, ...) を計算します。

nanones

与えられたテンソルと同じ形状でones値を持つテンソルを返し、Not a Numbers (NaNs) を保持します。

nanpca

与えられたデータに対して主成分分析を実行します。

nanshift

指定された次元に沿って配列要素をシフトし、NaNは無視します。

nanskew

各行の歪度を返し、非数値値（NaN）は無視します。

nanslope

`x`と`y`の相関係数の傾きを返し、NaN値は無視します。

nansum

Not a Numbers (NaNs) を無視して、すべての要素の合計を返します。

nanvar

各行の平均値を返し、Not a Numbers (NaNs) は無視します。

orthogonalize

指定された次元に沿って、xをyに対して直交化します。

orthonormalize

与えられたベクトルを正規直交化し、対応する正規直交ベクトルを返します。

pca

入力データ x に対して主成分分析を実行します。返される値は主成分を表します。具体的には、result[*, i, :] は * で指定されたバッチの \((i+1)\) 番目に大きい主成分を表します。形状: - x: \((*, S, D)\) ここで * は任意の追加次元数、S はセクション数、D は次元数を表します。qfeval では、次元数とセクション数はそれぞれシンボルとタイムスタンプを表すことが多いです。:returns:

project

与えられたテンソル x を、指定された射影行列 a を用いて投影します。

rand

[0, 1)の範囲の一様分布に従う乱数で埋められたテンソルを返します。

rand_like

入力と同じサイズのテンソルを返し、その値は[0, 1)の範囲の一様分布から生成された乱数で埋められます。

randint

低値（含む）から高値（除く）の範囲で一様に生成されたランダムな整数で埋められたテンソルを返します。

randn

平均0、分散1の正規分布に従う乱数で埋められたテンソルを返します（標準正規分布とも呼ばれます）。

randn_like

入力と同じサイズのテンソルを返し、その中に平均0、分散1の正規分布に従う乱数で埋めます。

randperm

`0`から`n - 1`までの整数をランダムに並べ替えた配列を返します。

rci

rcummax

次元 dim において、入力 input の要素の累積最大値を逆順に返します。

rcumsum

次元 dim において、input の要素の累積和を逆順に計算します。

rms

指定された次元 dim において、入力テンソルの各行の二乗平均平方根を返します。

rsi

定義 (use_sma=False): - https://www.investopedia.com/terms/r/rsi.asp - TA-lib定義と互換性あり (use_sma=True): - https://info.monex.co.jp/technical-analysis/indicators/005.html

shift

指定された次元に沿って配列要素をシフトします。

signif

指定された有効桁数にテンソルの数値を丸めます。

skipna

NaN値を除去した後、与えられたデータを指定された関数に適用します。

slope

`x`と`y`の相関関係の傾きを返します。

soft_topk_bottomk

与えられた次元と共に SoftTopKBottomK モジュールを適用します。

stable_sort

等価な要素の順序を保持しながら与えられたテンソルをソートします。
