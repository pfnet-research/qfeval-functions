qfeval_functions.functions.bollinger_band

bollinger_band(x, window=20, sigma=2.0, dim=-1)[ソース]

テンソルに対するボリンジャーバンドを計算します。

ボリンジャーバンドは、テクニカル分析ツールの一種で、中心バンド（移動平均線）と上下2本のバンドで構成されます。上下バンドは、中心バンドから sigma 標準偏差の距離に配置されます。この指標は金融分析において一般的に使用され、潜在的なサポートレベルやレジスタンスレベルを識別するために用いられます。

パラメータ:

  • x (Tensor) -- 時系列データを含む入力テンソル。

  • window (int, optional) -- 移動平均と標準偏差を算出するための移動ウィンドウのサイズ。デフォルト値は20です。

  • sigma (float, optional) -- 上バンドと下バンドの標準偏差の数。デフォルト値は2.0です。

  • dim (int, optional) -- ボリンジャーバンドを計算する次元。デフォルトは-1（最後の次元）です。

戻り値:

入力と同じ形状の3つのテンソルを含むタプル: - 上バンド: 中央バンド + (シグマ * 標準偏差) - 中央バンド: 移動平均 - 下バンド: 中央バンド - (シグマ * 標準偏差)

戻り値の型:

Tuple[Tensor, Tensor, Tensor]

サンプル

>>> x = torch.tensor([[100., 102., 101., 103., 105.],
...                   [50., 48., 49., 51., 52.]])
>>> upper, middle, lower = QF.bollinger_band(x, window=3, sigma=1.0, dim=1)
>>> upper
tensor([[     nan,      nan, 101.8165, 102.8165, 104.6330],
        [     nan,      nan,  49.8165,  50.5805,  51.9139]])
>>> middle
tensor([[     nan,      nan, 101.0000, 102.0000, 103.0000],
        [     nan,      nan,  49.0000,  49.3333,  50.6667]])
>>> lower
tensor([[     nan,      nan, 100.1835, 101.1835, 101.3670],
        [     nan,      nan,  48.1835,  48.0861,  49.4195]])
