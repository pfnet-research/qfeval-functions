qfeval_functions.functions.bincount

bincount(x, minlength=0, dim=-1)[ソース]

テンソル内の各値の出現回数をカウントします。

この関数は、指定された次元に沿って入力テンソル内の各非負整数値の出現頻度を計算します。出力テンソルの指定された次元方向のサイズは、minlength と入力テンソル内の最大値に 1 を加えた値の最大値となります。出力テンソルの各位置には、そのインデックス値の入力テンソル内での出現回数が格納されます。

パラメータ:

  • x (Tensor) -- 非負整数値を含む入力テンソル。

  • minlength (int, optional) -- カウント対象次元に沿った出力テンソルの最小長。 x 内の最大値が minlength - 1 未満の場合、出力はゼロでパディングされます。デフォルト値は 0 です。

  • dim (int, optional) -- カウントする次元。デフォルトは -1 で、最後の次元を意味します。

戻り値:

指定された次元に沿ってインデックス i の値が、入力テンソルの対応するスライス内で値 i が出現する回数を表しているテンソル。

戻り値の型:

Tensor

サンプル

>>> x = torch.tensor([1, 2, 2, 3, 3, 1])
>>> QF.bincount(x)
tensor([0, 2, 2, 2])

>>> x = torch.tensor([[1, 2, 2], [3, 3, 1]])
>>> QF.bincount(x, dim=1)
tensor([[0, 1, 2, 0],
        [0, 1, 0, 2]])

>>> x = torch.tensor([1, 2])
>>> QF.bincount(x, minlength=5)
tensor([0, 1, 1, 0, 0])
