qfeval_functions.functions.bincount

bincount(x, minlength=0, dim=-1)[ソース]

指定された次元に沿ってテンソル内の各値の出現頻度を計算します。出力テンソルの最小長を指定することも可能です。

戻り値の型:

Tensor

サンプル

>>> import torch
>>> x = torch.tensor([[1, 2, 2], [3, 3, 1]])
>>> bincount(x)
tensor([[0, 1, 2, 0],
        [0, 1, 0, 2]])

>>> x = torch.tensor([1, 2, 2, 3, 3, 1])
>>> bincount(x)
tensor([0, 2, 2, 2])
