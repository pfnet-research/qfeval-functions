qfeval_functions.functions.group_shift

group_shift(x, shift=1, dim=0, mask=None, refdim=-1, agg_f='any')[ソース]

指定された次元に沿ってテンソルをシフトし、指定されたマスクの位置はスキップします。

この関数はマスクで指定された位置に対してのみシフトを適用します。例えば、マスクが [True, False, True, False, True] の場合、[1, nan, 2, nan, 3] に対して1回シフトを適用すると [nan, nan, 1, nan, 2] という結果が得られます。

注意: シフトを適用する前に、マスクされていない値（つまりマスク値が False）は nan で埋められます。したがって、上記のマスク付きシフトを [1, 2, 3, 4, 5] に適用すると、[nan, nan, 1, nan, 3] という結果が得られます。ここで、マスクされていない値 (2, 4) は単に破棄されます。また、シフト値が 0 の場合 (shift == 0)、元の入力は復元されません。

パラメータ:

  • x (-) -- 入力テンソル。

  • shift (-) -- テンソルの要素がシフトされる位置の数。

  • dim (-) -- x がシフトされる次元。

  • mask (-) -- 1次元のブールテンソルで、`False`の値はシフト時にスキップするインデックスを指定します。テンソルの長さは`x.shape[dim]`と等しくなければなりません。

  • refdim (-) -- マスクが設定されている場合、マスクは自動的に生成されます。詳細は reduce_nan_patterns を参照してください。

戻り値:

x をシフトしたバージョン。

戻り値の型:

  • x_shifted
