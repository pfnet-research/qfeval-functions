qfeval_functions.functions.nanshift

nanshift(x, shift=1, dim=-1)[ソース]

Shift tensor elements along a dimension while preserving NaN positions.

この関数は、指定された次元に沿ってテンソルの有効な要素（NaNでない値）をシフトしつつ、NaN値を元の位置に保持します。標準的なシフト操作とは異なり、NaN値はシフト処理に参加しない「移動不可」の要素として扱われ、有効なデータのみがそれらの周囲にシフトされます。

これは、時系列データにおいて欠損値（NaNとして表現される）が元の時系列位置を保持したまま、有効な観測値が分析目的でシフトされる必要がある場合に特に有用です。

パラメータ:

  • x (Tensor) -- シフト対象の入力テンソル。

  • shift (int, optional) -- シフトする位置の数。正の値はインデックスの高い方向へ、負の値はインデックスの低い方向へシフトします。デフォルト値は1です。

  • dim (int, optional) -- シフトを実行する次元。デフォルトは-1（最後の次元）です。

戻り値:

入力と同じ形状のテンソルで、指定された次元に沿って有効な要素がシフトされ、NaNの位置は変更されずにそのまま保持されます。

戻り値の型:

Tensor

サンプル

>>> # Simple 1D shift with NaN values
>>> x = torch.tensor([1.0, nan, 3.0, 4.0, nan])
>>> QF.nanshift(x, shift=1)
tensor([nan, nan, 1., 3., nan])

>>> # Negative shift
>>> x = torch.tensor([1.0, nan, 3.0, 4.0, nan])
>>> QF.nanshift(x, shift=-1)
tensor([3., nan, 4., nan, nan])

>>> # 2D tensor shift along rows
>>> x = torch.tensor([[1.0, nan, 3.0],
...                   [4.0, 5.0, nan],
...                   [nan, 8.0, 9.0]])
>>> QF.nanshift(x, shift=1, dim=0)
tensor([[nan, nan, nan],
        [1., nan, nan],
        [nan, 5., 3.]])

>>> # 2D tensor shift along columns
>>> x = torch.tensor([[1.0, nan, 3.0],
...                   [4.0, 5.0, nan]])
>>> QF.nanshift(x, shift=1, dim=1)
tensor([[nan, nan, 1.],
        [nan, 4., nan]])

>>> # Large shift (wraps around valid elements)
>>> x = torch.tensor([1.0, nan, 3.0, 4.0, nan])
>>> QF.nanshift(x, shift=2)
tensor([nan, nan, nan, 1., nan])

>>> # All NaN tensor (no change)
>>> x = torch.tensor([nan, nan, nan])
>>> QF.nanshift(x, shift=1)
tensor([nan, nan, nan])

警告

シフト操作は有効な要素を循環させます。例えば、有効な要素が3つあり``shift=1``の場合、最後の有効な要素が最初の要素となり、他の要素は1つシフトします。

参考
