qfeval_functions.functions.fillna

fillna(x, nan=0.0, posinf=inf, neginf=-inf)[ソース]

Replace NaN and infinity values with specified numbers.

この関数は、テンソル内の NaN (Not a Number) 値、正の無限大値、負の無限大値をユーザーが指定した値で置換します。デフォルトでは、NaN 値は 0 に置換され、無限大値は保持されます。この動作は、デフォルトで無限大値を表現可能な最大/最小の有限値に置換する「torch.nan_to_num」関数とは異なります。

パラメータ:

  • x (Tensor) -- 置換対象の値を含む入力テンソル。

  • nan (float, optional) -- NaN を置き換える値。デフォルトは 0.0 です。

  • posinf (float, optional) -- 正の無限大を置き換える値。デフォルトは``math.inf``（正の無限大を保持します）。

  • neginf (float, optional) -- 負の無限大を置き換える値。デフォルトは -math.inf で、負の無限大をそのまま保持します。

戻り値:

入力と同じ形状とデータ型を持つ新しいテンソルで、NaN値と無限大値は指定されたパラメータに従って置換されます。

戻り値の型:

Tensor

サンプル

>>> # Replace NaN values with 0 (default behavior)
>>> x = torch.tensor([1.0, nan, 3.0, nan])
>>> QF.fillna(x)
tensor([1., 0., 3., 0.])

>>> # Replace NaN with -1
>>> x = torch.tensor([[nan, 2.0], [3.0, nan]])
>>> QF.fillna(x, nan=-1.0)
tensor([[-1.,  2.],
        [ 3., -1.]])

>>> # Handle infinity values
>>> x = torch.tensor([1.0, inf, -inf, nan])
>>> QF.fillna(x, nan=0.0, posinf=999.0, neginf=-999.0)
tensor([   1.,  999., -999.,    0.])

>>> # Preserve infinity by default
>>> x = torch.tensor([inf, -inf, nan])
>>> QF.fillna(x)
tensor([inf, -inf, 0.])
