qfeval_functions.functions.nanslope

nanslope(x, y, dim=(), keepdim=False)[ソース]

Compute the slope of simple linear regression between two tensors, ignoring NaN values.

この関数は、x と y の関係を最もよく説明する線形回帰直線の傾き (ベータ係数) を計算します。計算時には NaN 値は除外されます。傾きは、x の単位変化に対する y の変化率を表します。

この関数は、単純線形回帰における傾き係数を求めるための標準的な最小二乗法（OLS）の計算式を実装しています：

\[\beta = \frac{\text{共分散}(X, Y)}{\text{分散}(X)} = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}\]

ここで \(\bar{x}\) と \(\bar{y}\) はそれぞれ x と y の平均値であり、有効なデータペア (NaN を含まない) のみを用いて計算されます。

いずれかの x または y に特定の位置で NaN 値が存在する場合、そのペアは平均計算および傾き推定を含むすべての計算から除外されます。

パラメータ:

  • x (Tensor) -- 独立変数テンソル（予測子）。

  • y (Tensor) -- 従属変数テンソル（応答変数）。 :attr:`x`とブロードキャスト可能である必要があります。

  • dim (int or tuple of ints, optional) -- 傾きを計算する次元を指定します。指定しない場合（デフォルトは空のタプル）、全ての次元にわたって傾きが計算されます。

  • keepdim (bool, optional) -- 出力テンソルが dim を保持するかどうか。デフォルトは False。

戻り値:

有効なペア（NaNでない値）のみを用いて計算された傾き係数です。有効なペアが不足する場合や、x`の分散がゼロの場合、結果にNaN値が含まれる可能性があります。形状は入力次元、:attr:`dim、および:attr:`keepdim`パラメータに依存します。

戻り値の型:

Tensor

サンプル

>>> # Simple linear relationship with some NaN values
>>> x = torch.tensor([1.0, 2.0, nan, 4.0, 5.0])
>>> y = torch.tensor([2.0, 4.0, 6.0, nan, 10.0])
>>> QF.nanslope(x, y)
tensor(2.)

>>> # 2D tensors with slopes along rows
>>> x = torch.tensor([[1.0, 2.0, 3.0],
...                   [1.0, nan, 3.0]])
>>> y = torch.tensor([[2.0, 4.0, 6.0],
...                   [3.0, 5.0, nan]])
>>> QF.nanslope(x, y, dim=1)
tensor([2., nan])

>>> # Perfect linear relationship
>>> x = torch.tensor([1.0, 2.0, 3.0, 4.0])
>>> y = torch.tensor([3.0, 5.0, 7.0, 9.0])  # y = 2x + 1
>>> QF.nanslope(x, y)
tensor(2.)

>>> # With keepdim
>>> x = torch.tensor([[1.0, nan, 3.0],
...                   [2.0, 4.0, 6.0]])
>>> y = torch.tensor([[2.0, 4.0, nan],
...                   [1.0, 2.0, 3.0]])
>>> QF.nanslope(x, y, dim=1, keepdim=True)
tensor([[   nan],
        [0.5000]])

>>> # Broadcasting example
>>> x = torch.tensor([[1.0], [2.0], [3.0]])
>>> y = torch.tensor([2.0, 4.0, 6.0])
>>> QF.nanslope(x, y)
tensor(0.)

>>> # Zero variance in x (undefined slope)
>>> x = torch.tensor([2.0, 2.0, 2.0])
>>> y = torch.tensor([1.0, 3.0, 5.0])
>>> QF.nanslope(x, y)
tensor(nan)

警告

傾きの計算には少なくとも2組の有効な（NaNでない）観測値ペアが必要です。有効なペアが2組未満の場合、結果はNaNになります。さらに、すべての有効な:attr:`x`値が同一である場合（分散が0の場合）、数学的に傾きは定義されません。

参考
