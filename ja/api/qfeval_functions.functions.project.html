qfeval_functions.functions.project

project(a, x)[ソース]

指定された投影行列 a を使用して、与えられたテンソル x を投影します。

パラメータ:

  • a (Tensor) -- 射影行列。

  • x (Tensor) -- 投影対象のテンソル。

戻り値の型:

Tensor

Returns

y (Tensor): 射影されたテンソル。

形状:
  • a: \((*, O, I)\) 形式の行列。ここで \(*\) は任意の数の次元を表します。

    次元数（Noneを含む）、:math:`O`は出力次元数、:math:`I`は入力次元数を表す。

  • x: \((*, S, I)\) 形式のテンソル。ここで \(*\) は任意の数の次元を表します。

    次元数（Noneを含む）、:math:`S`はセクション数、:math:`I`は入力次元数を表す。

  • return: \((*, S, O)\), where \(*\) represents any number of

    次元数（Noneを含む）、:math:`S`はセクション数、:math:`O`は出力次元数を表す。

qfevalにおいて、I/O次元とセクションはそれぞれシンボルとタイムスタンプを表すことが多い。
