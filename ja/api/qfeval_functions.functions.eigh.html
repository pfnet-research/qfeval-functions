qfeval_functions.functions.eigh

eigh(tensor, uplo='L')[ソース]

Compute eigenvalues and eigenvectors of a symmetric/Hermitian matrix.

この関数は、実対称行列または複素エルミート行列の固有値と固有ベクトルを計算します。固有値は昇順に返され、対応する固有ベクトルは正規化されます。この実装では、CPUテンソルにはNumPyを、CUDAテンソルにはCuPyを使用しています。

注釈

This function does not support automatic differentiation (autograd).

パラメータ:

  • tensor (Tensor) -- 形状が (..., N, N) の対称行列（実数の場合）またはエルミート行列（複素数の場合）。 uplo パラメータに応じて、上三角部分または下三角部分が使用されます。

  • uplo (str, optional) -- 使用する行列の三角部分を示します： - 'L' または 'l': 下三角部分を使用する（デフォルト） - 'U' または 'u': 上三角部分を使用する

戻り値:

以下の要素を含むタプル: - 固有値: 昇順に並んだ固有値を含む形状 (..., N) のテンソル - 固有ベクトル: 形状 (..., N, N) のテンソルで、列は対応する固有値の正規化された固有ベクトル

戻り値の型:

Tuple[Tensor, Tensor]

サンプル

>>> # Create a symmetric matrix
>>> A = torch.tensor([[4.0, -2.0],
...                   [-2.0, 3.0]])
>>> eigenvalues, eigenvectors = QF.eigh(A)
>>> eigenvalues
tensor([1.4384, 5.5616])

>>> # Verify: A @ v = λ @ v
>>> torch.allclose(A @ eigenvectors, eigenvectors @ torch.diag(eigenvalues))
True

>>> # Using upper triangular part
>>> B = torch.tensor([[1.0, 2.0, 3.0],
...                   [0.0, 4.0, 5.0],
...                   [0.0, 0.0, 6.0]])
>>> eigenvalues, eigenvectors = QF.eigh(B, uplo='U')
