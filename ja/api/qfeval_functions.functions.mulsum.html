qfeval_functions.functions.mulsum

mulsum(x, y, dim=(), keepdim=False, mean=False, *, _ddof=0)[ソース]

メモリ効率に優れた方法で、要素ごとの積の和または平均を計算します。

この関数は、中間的な積テンソルをメモリ上に作成することなく、2つのテンソルの要素ごとの積の和（または平均） (x * y).sum(dim) または (x * y).mean(dim) を計算します。これは、特に大規模なテンソルを扱う場合や、ブロードキャスト操作によって中間テンソルが非常に大きくなり、利用可能なメモリを超える可能性がある場合に特に重要です。

パラメータ:

  • x (Tensor) -- 最初の入力テンソル。

  • y (Tensor) -- 第2の入力テンソル。 :attr:`x`とブロードキャスト可能である必要があります。

  • dim (int or tuple of ints, optional) -- 和または平均を計算する次元を指定します。指定しない場合（デフォルトは空のタプル）、全ての次元にわたって計算が行われます。

  • keepdim (bool, optional) -- 出力テンソルが dim を保持するかどうか。デフォルトは False。

  • mean (bool, optional) -- Trueの場合、和ではなく平均を計算します。デフォルトはFalse（和を計算）です。

  • _ddof (int, optional) -- 平均計算における自由度の増分。使用する除数は N - _ddof で、ここで N は要素数を表します。デフォルト値は 0 です。これは内部パラメータです。

戻り値:

要素ごとの積の和または平均。形状は入力次元、dim、および keepdim パラメータに依存します。

戻り値の型:

Tensor

サンプル

>>> # Simple element-wise product sum
>>> x = torch.tensor([1.0, 2.0, 3.0])
>>> y = torch.tensor([2.0, 3.0, 4.0])
>>> QF.mulsum(x, y)
tensor(20.)

>>> # Equivalent to (x * y).sum()
>>> torch.allclose(QF.mulsum(x, y), (x * y).sum())
True

>>> # Mean instead of sum
>>> QF.mulsum(x, y, mean=True)
tensor(6.6667)

>>> # 2D tensors with specific dimension
>>> x = torch.tensor([[1.0, 2.0], [3.0, 4.0]])
>>> y = torch.tensor([[2.0, 1.0], [1.0, 2.0]])
>>> QF.mulsum(x, y, dim=1)
tensor([ 4., 11.])

>>> # Memory-efficient broadcasting
>>> x = torch.randn(1000, 1)
>>> y = torch.randn(1, 1000)
>>> # Efficiently computes without creating 1000x1000 intermediate tensor
>>> result = QF.mulsum(x, y)

>>> # With keepdim
>>> x = torch.tensor([[[1.0, 2.0]], [[3.0, 4.0]]])
>>> y = torch.tensor([[[2.0, 1.0]], [[1.0, 2.0]]])
>>> QF.mulsum(x, y, dim=(1, 2), keepdim=True)
tensor([[[ 4.]],

        [[11.]]])
